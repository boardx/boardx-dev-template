# Board Menu Use Cases

Board Menu 是用户在可编辑 Board 中选择工具、切换模式和插入内容的入口。

## 用户入口

- 画布边缘的工具栏。
- 工具子菜单、资源面板、模板面板、文件拖放或条件性入口。

## 权限口径

- Owner/Admin/Member 在具备编辑权限时可以使用创建和编辑能力。
- Visitor 与 public visitor 以查看、导航和允许的导出/复制能力为主，不能改变 Board 内容或访问策略。
- 分享范围以 public、team、room、board 等 Board 访问策略和当前用户在 Room/Team 中的身份共同决定。

## 业务规则

- 当前确认的主工具栏入口包括选择、平移、便利贴、手绘/橡皮、文本、连接线、形状、资源和模板。
- 图表、独立文件、链接、AI 入口以当前产品开关和界面实际展示为准。

## 仍需产品确认

- Chart、File、Link、AI 是否作为主工具栏独立按钮对所有用户开放。
