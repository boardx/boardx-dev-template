# Board Header Use Cases

本目录描述用户进入 Board 后通过 Header 查看状态并进入顶部功能。

## 用户入口

- 顶部返回、标题、撤销/重做、语音、模式切换、同步状态、在线成员、分享、跟随、计时器、Slides 和更多菜单。
- 窄屏下折叠后的 Header 入口。

## 权限口径

- Owner/Admin/Member 在具备编辑权限时可以使用创建和编辑能力。
- Visitor 与 public visitor 以查看、导航和允许的导出/复制能力为主，不能改变 Board 内容或访问策略。
- 分享范围以 public、team、room、board 等 Board 访问策略和当前用户在 Room/Team 中的身份共同决定。

## 业务规则

- Header 中状态可见不代表可编辑。
- 分享范围变更需要管理权限；复制链接对更多角色可见时也不代表可改访问策略。
