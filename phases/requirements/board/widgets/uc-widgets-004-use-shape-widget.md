Use Case 名称：
使用形状组件

Actor：
Board Owner、Board Admin、Board Member

目标：
用户在 Board 中创建形状，用形状承载结构化内容，并调整形状的文本、外观、大小和层级。

系统边界：
BoardX / CanvasX Widgets

前端入口：
1. Board 主工具栏中的形状入口。
2. 形状入口旁的形状类型下拉菜单。
3. 画布上已有形状。
4. 选中形状后出现的 Widget Menu。
5. 对形状右键或打开更多菜单后的 Context Menu。

前置条件：
1. 用户已打开有编辑权限的 Board。
2. 目标组件类型在当前 Board 中可用。

触发条件：
用户选择形状工具，或选中画布上的已有形状。

主流程：
1. 用户打开形状入口，系统展示当前可创建的形状类型下拉菜单。
2. 系统当前确认展示的形状类型包括圆形、三角形、菱形、圆角矩形、矩形和六边形。
3. 用户选择一种形状，系统把该形状设为当前形状工具，并在形状按钮上显示对应图标。
4. 用户再次使用形状工具时，系统默认沿用上次选择的形状类型。
5. 用户在画布上按下并拖拽，系统按拖拽范围创建所选类型的形状，并显示选中框、缩放控制点和可编辑区域；尺寸过小时不保留新形状。
6. 用户拖拽形状本体，系统移动形状位置。
7. 用户拖拽边角或控制点，系统改变形状大小。
8. 用户输入或修改形状内文本，系统在形状内展示文字，并根据形状尺寸重新排版。
9. 用户调整背景色，系统更新形状填充颜色。
10. 用户调整边框颜色和边框粗细，系统更新形状轮廓显示。
11. 用户调整字体颜色、字体、字号、加粗和文本对齐方式，系统更新形状内文本样式。
12. 用户通过连接线入口把形状与其它对象连接，系统显示连接点或连接反馈，并保留连接关系。
13. 用户通过 Context Menu 对形状执行复制、剪切、粘贴、删除、锁定、解锁、置顶置底、编组、取消编组、对齐或保存为模板等动作。
14. 用户点击 AI 辅助入口时，系统以形状文本或选中内容作为上下文进入 AI 辅助流程。
15. 操作完成后，形状留在画布中，并参与协作同步、撤销重做、选择、多选和导出。

备选流程：
1. 用户只创建空形状时，系统保留空形状，用户稍后仍可输入文本。
2. 用户只切换形状类型但不在画布创建时，系统仅更新当前形状工具状态，不创建新对象。
3. 用户选中已有形状时，系统允许修改内容、颜色、边框、字体、位置、大小和层级；已有形状类型切换只有在 Widget Menu 明确展示该入口时才可操作。
4. 多选形状时，系统进入多选状态，只展示多选适用的移动、对齐、分布、锁定、删除、编组等动作。
5. 形状被锁定时，系统允许查看，但不允许移动、缩放、改色、改文本或删除。

异常流程：
1. 用户没有编辑权限时，系统不创建形状，也不允许修改已有形状。
2. AI 辅助失败时，系统保留原形状内容和样式。
3. 协作过程中形状被其他用户删除或锁定时，系统关闭当前编辑状态并刷新画布。

权限与可见性：
1. Board Owner：可以进入 Board、编辑内容、管理标题和内容生命周期，并可在当前 Board、Room、Team 权限允许范围内管理分享范围、删除、移动、复制等高风险操作。
2. Board Admin：可以编辑内容并使用多数协作入口；涉及可见范围、删除、移动、备份恢复等管理动作时，需要具备对应 Room/Team/Board 管理权限。
3. Board Member：可以在有编辑权限的 Board 中创建、修改、移动、复制和删除自己可操作的内容；不默认拥有权限策略和 Board 生命周期管理能力。
4. Board Visitor：可以查看允许访问的内容，使用平移、缩放、查看同步状态和在线状态等查看能力；不能创建、编辑、删除或改变分享范围。
5. Public visitor：通过公开链接进入时只能获得公开策略允许的查看能力；如需要保存、编辑、加入协作或访问非公开 Board，系统应引导登录、加入空间或提示无权限。

后置条件：
1. 当前形状工具类型被切换，或形状被创建、修改、移动、缩放、连接、锁定、删除，或系统明确提示当前动作不可用。

不包含：
1. 不描述系统内部实现。
2. 不描述系统内部存储和同步机制。
3. 不把已存在的实现模块等同于当前界面必然显示的入口。

业务规则：
1. 组件能力由对象类型、锁定状态、选中数量和角色权限决定。
2. Widget Menu 与 Context Menu 都不能绕过 Board 编辑权限。
3. 形状的类型、文字、尺寸、填充、边框、字体、位置和层级都属于用户可感知状态。
4. 形状内文字和形状本身应作为同一个对象参与选择、移动、缩放和权限控制。
5. 形状工具当前可选类型以界面实际展示为准；当前确认展示圆形、三角形、菱形、圆角矩形、矩形和六边形。
6. 已有形状类型切换由 Widget Menu 可见入口决定；未展示该入口时，用户不能直接把已有形状切换为另一种形状。
