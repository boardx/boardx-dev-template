Use Case 名称：
使用便利贴组件

Actor：
Board Owner、Board Admin、Board Member

目标：
用户在 Board 中创建便利贴，用便利贴记录想法，并对便利贴的内容、外观、位置和状态进行编辑。

系统边界：
BoardX / CanvasX Widgets

前端入口：
1. Board 左侧或底部主工具栏中的便利贴入口。
2. 画布上已有便利贴。
3. 选中便利贴后出现的 Widget Menu。
4. 对便利贴右键或打开更多菜单后的 Context Menu。

前置条件：
1. 用户已打开有编辑权限的 Board。
2. 目标组件类型在当前 Board 中可用。

触发条件：
用户选择便利贴工具，或选中画布上的已有便利贴。

主流程：
1. 用户点击便利贴工具，系统显示可创建便利贴的状态。
2. 用户在画布目标位置点击，系统在该位置创建默认尺寸的便利贴，并显示便利贴边界、选中框和可用菜单。
3. 用户输入或修改便利贴文本，系统在便利贴内显示文字内容，并根据便利贴尺寸进行排版。
4. 用户拖拽便利贴本体，系统移动便利贴位置，并在移动过程中显示当前位置反馈。
5. 用户拖拽便利贴边角或控制点，系统改变便利贴大小，文字区域随尺寸变化重新排布。
6. 用户通过 Widget Menu 切换便利贴形态，系统在矩形、圆形或形状便利贴等可用样式之间切换，并保留原文本内容。
7. 用户调整背景色，系统更新便利贴填充颜色。
8. 用户调整字体颜色、字体、字号、加粗和文本对齐方式，系统更新便利贴中的文字样式。
9. 用户调整边框或线条样式，系统更新便利贴边框的显示效果。
10. 用户使用应用格式能力时，系统把当前便利贴的可复用样式应用到用户后续指定的其它可兼容组件。
11. 用户选中便利贴后，可以通过拖拽、键盘移动、复制、剪切、粘贴、删除、锁定、解锁、置顶置底、编组或取消编组等通用组件操作管理便利贴。
12. 用户需要与其它内容建立关系时，可以从便利贴附近创建或连接连接线；系统显示连接反馈，并在连接成功后保留连接关系。
13. 用户点击 AI 辅助入口时，系统以当前便利贴内容作为上下文进入 AI 辅助流程；处理结果按用户确认后的动作更新或生成内容。
14. 操作完成后，便利贴停留在画布中，并参与协作同步、撤销重做、选择、多选和导出。

备选流程：
1. 用户双击已有便利贴时，系统进入文本编辑状态；用户点击画布空白处后结束编辑。
2. 用户只选中便利贴但不进入文本编辑时，系统允许移动、缩放、复制、删除和样式调整。
3. 多选便利贴时，系统进入多选状态，只展示多选适用的移动、对齐、锁定、删除、编组等动作。
4. 便利贴被锁定时，系统允许查看和选择受限状态，但不允许编辑文本、移动、缩放、改色或删除。

异常流程：
1. 用户没有编辑权限时，系统不创建便利贴，也不允许修改已有便利贴。
2. AI 辅助失败时，系统保留原便利贴内容和样式，并提示用户重试。
3. 协作过程中便利贴被其他用户删除或锁定时，系统关闭当前编辑状态并刷新画布。
4. 文本过长或字号过大导致显示受限时，系统仍保留文本内容，并按当前便利贴尺寸展示可见部分或自动排版。

权限与可见性：
1. Board Owner：可以进入 Board、编辑内容、管理标题和内容生命周期，并可在当前 Board、Room、Team 权限允许范围内管理分享范围、删除、移动、复制等高风险操作。
2. Board Admin：可以编辑内容并使用多数协作入口；涉及可见范围、删除、移动、备份恢复等管理动作时，需要具备对应 Room/Team/Board 管理权限。
3. Board Member：可以在有编辑权限的 Board 中创建、修改、移动、复制和删除自己可操作的内容；不默认拥有权限策略和 Board 生命周期管理能力。
4. Board Visitor：可以查看允许访问的内容，使用平移、缩放、查看同步状态和在线状态等查看能力；不能创建、编辑、删除或改变分享范围。
5. Public visitor：通过公开链接进入时只能获得公开策略允许的查看能力；如需要保存、编辑、加入协作或访问非公开 Board，系统应引导登录、加入空间或提示无权限。

后置条件：
1. 便利贴被创建、修改、移动、缩放、连接、锁定、删除，或系统明确提示当前动作不可用。

不包含：
1. 不描述系统内部实现。
2. 不描述系统内部存储和同步机制。
3. 不把已存在的实现模块等同于当前界面必然显示的入口。

业务规则：
1. 组件能力由对象类型、锁定状态、选中数量和角色权限决定。
2. Widget Menu 与 Context Menu 都不能绕过 Board 编辑权限。
3. 便利贴的文字内容、形态、尺寸、颜色、边框、字体和位置都属于用户可感知状态。
4. 便利贴切换形态不应丢失原有文本内容。
5. 锁定状态下不得通过快捷键、菜单或协作同步入口绕过编辑限制。
