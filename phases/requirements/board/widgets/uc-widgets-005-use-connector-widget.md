Use Case 名称：
使用连接线组件

Actor：
Board Owner、Board Admin、Board Member

目标：
用户在 Board 中用连接线表达对象之间的关系，并通过当前可见的线型、端点和路径样式让关系含义更清楚。

系统边界：
BoardX / CanvasX Widgets

前端入口：
1. Board 主工具栏中的连接线入口。
2. 画布对象附近的连接点或连接操作入口。
3. 画布上已有连接线。
4. 选中连接线后出现的 Widget Menu。
5. 对连接线右键或打开更多菜单后的 Context Menu。

前置条件：
1. 用户已打开有编辑权限的 Board。
2. 目标组件类型在当前 Board 中可用。

触发条件：
用户选择连接线工具，或从一个对象拖拽连接到另一个对象。

主流程：
1. 用户选择连接线工具，系统进入连接线创建状态。
2. 用户在画布空白处拖拽，系统创建自由连接线，并显示起点、终点和路径反馈。
3. 用户从一个对象的连接点拖拽到另一个对象，系统高亮可连接目标，并在释放后建立对象间连接关系。
4. 用户选中连接线，系统显示连接线选中状态、端点控制点和可用的连接线样式菜单。
5. 用户拖拽连接线起点或终点，系统改变连接对象或端点位置。
6. 用户拖拽连接线路径或控制点，系统改变连接线走向。
7. 用户调整线条颜色和线宽，系统更新连接线外观。
8. 用户打开连接线样式菜单，系统展示当前可用的线宽、路径形态和端点样式入口；颜色等其它入口只有实际显示时才可操作。
9. 用户调整线条颜色和线宽时，系统立即更新连接线外观。
10. 用户调整路径形态，系统在当前可用的直线、曲线等路径之间切换，并保持原起点和终点。
11. 用户调整连接线端点样式，系统按当前可见选项更新箭头或无箭头状态。
12. 当连接线样式菜单提供实线/虚线、折线或文字标识入口时，用户才可以操作这些能力；未展示入口不能写成当前必然可用。
13. 用户通过 Context Menu 对连接线执行复制、剪切、粘贴、删除、锁定、解锁、置顶置底等动作。
14. 操作完成后，连接线留在画布中，并在被连接对象移动时保持可理解的关系表达。

备选流程：
1. 用户只连接到空白画布时，系统保留自由端点连接线。
2. 用户移动被连接对象时，系统保持连接线跟随或刷新连接关系。
3. 用户没有输入标识时，系统只显示连接线本身，不强制显示默认文字。
4. 连接线文字标识入口未显示时，系统只显示连接线本身，不提供标识编辑。
5. 多选连接线和其它对象时，系统进入多选状态，只展示多选适用动作。
6. 连接线被锁定时，系统允许查看，但不允许拖动端点、改样式、改标识或删除。

异常流程：
1. 用户没有编辑权限时，系统不创建连接线，也不允许修改已有连接线。
2. 目标对象被删除时，系统移除连接关系或把连接线端点保留为自由端点。
3. 协作过程中连接线被其他用户删除或锁定时，系统关闭当前编辑状态并刷新画布。

权限与可见性：
1. Board Owner：可以进入 Board、编辑内容、管理标题和内容生命周期，并可在当前 Board、Room、Team 权限允许范围内管理分享范围、删除、移动、复制等高风险操作。
2. Board Admin：可以编辑内容并使用多数协作入口；涉及可见范围、删除、移动、备份恢复等管理动作时，需要具备对应 Room/Team/Board 管理权限。
3. Board Member：可以在有编辑权限的 Board 中创建、修改、移动、复制和删除自己可操作的内容；不默认拥有权限策略和 Board 生命周期管理能力。
4. Board Visitor：可以查看允许访问的内容，使用平移、缩放、查看同步状态和在线状态等查看能力；不能创建、编辑、删除或改变分享范围。
5. Public visitor：通过公开链接进入时只能获得公开策略允许的查看能力；如需要保存、编辑、加入协作或访问非公开 Board，系统应引导登录、加入空间或提示无权限。

后置条件：
1. 连接线被创建、改线型、改路径、改端点、改颜色、改粗细、锁定、删除，或系统明确提示当前动作不可用。

不包含：
1. 不描述系统内部实现。
2. 不描述系统内部存储和同步机制。
3. 不把已存在的实现模块等同于当前界面必然显示的入口。

业务规则：
1. 组件能力由对象类型、锁定状态、选中数量和角色权限决定。
2. Widget Menu 与 Context Menu 都不能绕过 Board 编辑权限。
3. 连接线的起点、终点、连接对象、路径、线条样式、颜色、线宽和端点样式都属于用户可感知状态。
4. 连接线不能绕过被连接对象的锁定和权限边界。
5. 连接线菜单能力以当前 Widget Menu 实际展示为准；未展示的实线/虚线、折线和文字标识入口不能写成当前必然可用。
