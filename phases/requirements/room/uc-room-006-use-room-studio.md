Use Case 名称：
使用房间 Studio 面板

Actor：
Room member、Room admin、Room owner

目标：
用户在房间聊天工作区打开右侧 Studio 面板，从房间文件生成音频概览、思维导图、报告、闪卡或数据表等 AI 产物。

系统边界：
BoardX / Room Studio

前端入口：
房间 Chat 页签中的聊天工作区右侧 Studio 面板。

前置条件：
- 用户已登录并已进入目标房间。
- 用户已进入某个房间聊天线程。

触发条件：
用户打开房间聊天线程。

主流程：
1. 用户进入房间 Chat 页签。
2. 用户新建或打开一个聊天线程后，系统展示三栏工作区。
3. 右侧展示 Room Studio 面板，用户看到 Audio Overview、Mind Map、Report、Flashcards、Data Table 等工具卡片。
4. 用户点击某个工具，系统打开文件选择侧边面板，并展示该工具说明。
5. 如果当前聊天线程没有文件，系统提示先在 Files 上传文件。
6. 如果有文件，用户可勾选一个或多个文件；已选文件以选中样式显示。
7. 用户点击 Generate，系统显示生成中状态，并禁用重复生成。
8. 生成成功后，面板展示结果内容，并显示来源文件数量。
9. 用户可以点击 Edit Selection 返回文件选择，或点击 Close 关闭面板。
10. 用户可以收起 Studio 面板；收起后页面保留展开按钮。
11. 用户点击展开按钮，系统重新显示 Studio 面板。
12. 用户可以拖动右侧分隔条调整 Studio 面板宽度。
13. 中间聊天区域会根据右侧面板宽度预留空间，避免遮挡聊天内容。

备选流程：
- A1：用户保持 Studio 收起，只使用聊天区。
- A2：用户调整 Studio 宽度后继续对话。
- A3：用户打开工具后未选择文件，Generate 保持不可用。

异常流程：
- E1：如果聊天线程被删除，系统返回房间 Chat 列表。
- E2：生成失败时，文件选择面板显示 Generation failed 或后端返回的错误信息。

权限与可见性：
- owner/admin/member 在进入房间聊天线程后均可看到该布局。
- 非房间成员无法进入房间工作区。
- visitor/未登录用户不能进入房间 Studio 面板。

后置条件：
- 用户可以在聊天工作区中看到或隐藏 Studio 面板。
- 生成成功时，用户在 Studio 侧边面板中看到 AI 产物结果。

不包含：
- 不包含产物导出、保存到白板或二次编辑流程。

业务规则：
- Studio 工具基于当前聊天线程可见的房间文件生成；没有文件时不能生成。
