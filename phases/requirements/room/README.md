# Room 用例

本目录描述用户在当前团队下创建、查看、进入和管理房间的场景。

覆盖范围：
- 创建房间。
- 查看、搜索、收藏并打开房间。
- 邀请与管理房间成员。
- 更新或删除房间。
- 在房间 Chat 工作区查看文件面板与 Studio 面板。
- 进入房间 Survey 页签。

角色边界：
- Room owner/admin 可以看到房间设置入口，并管理成员、名称和删除流程。
- Room member 可以进入自己已加入的房间，使用 Board、Chat、Survey 等内容入口。
- 非房间成员直接进入房间详情会被提示无权访问并返回房间列表。

覆盖审计提示集中写入各具体用例的“业务规则”字段。
