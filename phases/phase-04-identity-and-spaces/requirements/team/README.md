# Team 用例

本目录描述用户在 BoardX 中创建、切换和管理团队的真实使用场景。

覆盖范围：
- 创建团队、查看并切换团队。
- 通过邮箱或邀请链接邀请成员。
- 通过邀请链接加入团队。
- 管理成员、角色、token 权限和用量。
- 更新团队名称/头像、删除团队。
- 进入 Team 管理页、Memory、AI Store 等团队级入口。

角色边界：
- owner/admin 可以进入 Team 管理入口。
- member 可以切换和使用自己已加入的团队，但进入 Team 管理入口时会收到无权限提示。
- 通过邀请加入的用户默认按成员处理，后续角色调整由管理者完成。

覆盖审计提示集中写入各具体用例的“业务规则”字段。
